# 如何运行多个 ZFile 实例

请结合文档中这两个方法：

1. 找到配置文件所在路径，可参考：<a target="_blank" href="/config/config-path">配置文件路径</a>
2. 修改配置文件中的端口，让它们不要冲突，可参考：<a target="_blank" href="/config/config-port">配置文件-启动端口</a>
3. 修改配置文件中的数据库路径，让它们不要冲突，可参考：<a target="_blank" href="/config/config-db">配置文件-数据库配置</a>

:::caution
注意，所有配置修改后都需要重启 ZFile 才能生效。
:::
